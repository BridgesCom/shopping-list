# Список покупок (PWA)

Сімейний список покупок як Progressive Web App - ставиться на iPhone іконкою, без App Store.

## Етапи розробки

- [x] **Етап 1** - каркас PWA (локальне зберігання), GitHub Pages, встановлення на iPhone
- [ ] **Етап 2** - синхронізація через Firebase Firestore (спільний список, realtime)
- [ ] **Етап 3** - вхід через Google + приватність (доступ лише для учасників)
- [ ] **Етап 4** - запрошення родини

## Структура

- `index.html` - сам додаток (розмітка, стилі, логіка)
- `manifest.json` - маніфест PWA (іконка, назва, повноекранний режим)
- `service-worker.js` - офлайн-кеш
- `icons/` - іконки додатка
- `gen_icons.py` - генератор іконок (Pillow); запуск: `python gen_icons.py`

## Локальний запуск

```
python -m http.server 8000
```

Потім відкрити http://localhost:8000

## Оновлення

Після зміни кешованих файлів збільшити версію кешу в `service-worker.js`
(`spysok-v1` -> `spysok-v2`), щоб пристрої підхопили нову версію.
